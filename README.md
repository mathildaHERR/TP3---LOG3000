## Nom du projet 
Calculatrice simple

## Numéro d'équipe 
8

## Objectif
Le but du projet est de fournir à un utilisateur une calculatrice simple. Celle-ci sera disponible comme application web. La calculatrice permet à un utilisateur de sélectionner deux nombres et une opération à éffectuer sur ceux-ci. Le résultat est montré sur la calculatrice.

## Prérequis d'installation
### Avoir python d'installé

### Avoir pip d'installé

```python get-pip.py```

### Avoir Flask d'installé

```python python -m pip install -U flask```

## Instructions d'installation
1. Installer les prérequis si ce n'est pas encore fait à l'aide d'un terminal.
2. Cloner le projet sur son ordinateur avec GitHub en copiant le lien HTTPS disponible sur GitHub.
3. Ouvrir une IDE qui supporte python.
4. Trouver l'option pour clôner un repositoire Git.
5. Insérer l'URL copiée depuis GitHub.
6. Cliquer sur clôner l'URL.
7. Sélectionner un endoit dans son ordinateur pour mettre le fichier clôné.

## Instructions d'utilisation
1. Ouvrir le projet sur une IDE qui supporte python ou ouvrir son terminal à l'emplacement du projet.
2. Dans le terminal, écrire la commande: ```python python app.py ```.
3. Si vous n'êtes pas redirigé à la page web, suivre sur le lien où il est écrit que l'application est déployée.
4. Pour effectuer une opération mathématique: écrire un nombre, suivi par un opérateur, suivi par un autre nombre et cliquer sur le bouton ```=```.
5. Pour effacer l'écran de la calculatrice: cliquer sur le bouton ```C```.
6. Pour arrêter l'application: effectuer ```ctrl + c``` dans le terminal où vous avez déployé l'application.
