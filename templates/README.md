## Raison d'être du module
La raison d'être du module `templates` est de contenir les fichiers qui servent de base à créer d'autre fichiers similaires plus simplement et rapidement.

## Les principaux fichiers
Le fichier principal est `index.html`. La responsabilité de ce fichier est de créer un template pour l'interface de la calculatrice. Cette interface peut alors être modifiée, dépendant de si il y a un nombre, une expression ou un résultat sur l'écran de la calculatrice.

## Dépendaces ou hypothèses
Le fichier `index.html` dans le module `templates` est dépendant du fichier `style.css` dans le module `static`.
